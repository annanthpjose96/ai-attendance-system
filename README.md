# AI Attendance System

An AI-powered attendance management system that automates attendance tracking using face recognition and voice verification technologies.

## Features

- Face Recognition Attendance
- Voice Verification
- Student Management
- Teacher Dashboard
- Attendance Tracking
- Attendance Reports

## Technologies Used

- Python
- Streamlit
- OpenCV
- Face Recognition
- Speech Recognition
- SQLite

## Project Structure

```
src/
├── components/
├── database/
├── pipelines/
├── screens/
└── ui/
```

## Installation

```bash
pip install -r requirements.txt
python app.py
```
